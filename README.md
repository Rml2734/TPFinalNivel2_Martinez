Gestión de Artículos - Nivel 2 C#

Este es mi proyecto final para el Nivel 2 del curso de C#. Consiste en una aplicación de escritorio (WinForms) para la administración de un catálogo de artículos.

Funcionalidades

Listado Completo: Visualización de todos los artículos en una grilla.

Gestión (ABML): Alta, Baja, Modificación y Lectura de detalles.

Filtros Avanzados: Búsqueda por múltiples campos (Nombre, Precio, Categoría, etc.).

Validaciones: Control de errores de usuario y tipos de datos.

Arquitectura

El proyecto fue desarrollado utilizando una arquitectura en 3 capas:

Dominio: Clases de entidades.

Negocio: Lógica de acceso a datos y filtros.

Presentación: Interfaz de usuario (Windows Forms).

Base de Datos

Utiliza SQL Server con la base de datos CATALOGO_DB.
